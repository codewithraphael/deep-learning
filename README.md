# Deep Learning Projects

This repository features focused deep learning projects built around real datasets and hands-on model development.
Each project includes data preparation, model implementation, evaluation, and notes on what was learned.

The work here is split between:

- from-scratch model implementations for core neural network concepts
- practical dataset experiments in image and text domains
- end-to-end workflows that combine preprocessing, training, and analysis

## What’s Included

- `computer-vision/` — image-based deep learning experiments and model code
- `natural-language-processing/` — NLP projects for text classification and sentiment analysis
- `neural-networks/` — foundational neural network implementations with minimal dependencies

## Built With

- Python
- NumPy / pandas
- Tensorflow
- Keras
- Scikit-learn
- Notebook-based project exploration
- Visualization libraries for analysis and model interpretation

## Purpose

Use this repository to:

- practice building deep learning models from first principles
- explore real data through preprocessing and feature analysis
- compare model behavior on applied tasks like spam detection and classification
- document experiments with reproducible project structure
